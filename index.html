Example Webpack Application

This application contain a setup with webpack including many features, the idea is to combine multiple tools to learn how to use webpack for complex projects with multiple frameworks and languages.

Features

Frameworks

  • React 18
  • Vue 3

Styles

  • Sass
  • Css

Languages

  • JavaScript
  • TypeScript
  • JSX/TSX

Optimizations

  • Code Splitting
  • Lazy Loading
  • Minifying (With default webpack terser plugin)
  • Css minifying
  • Css extraction (serve css in its own file)
  • Compressing (With gzip)

Development and Debugging Features

  • Hot Module Replacement
  • React Refresh (Keep State on HMR)
  • Source Maps

PWA Support

  • Service Worker (With Workbox Defaults)
  • Installation
